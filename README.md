# Лабораторна робота №18 (Варіант 26)

**Тема:** База даних спортивного змагання (легка атлетика).

Проєкт адаптовано під **Dev-C++ 5.11** (GNU C++11) і консольний режим. Для Windows у `main.cpp` додано налаштування кирилиці через:
- `SetConsoleCP(1251);`
- `SetConsoleOutputCP(1251);`

## Виконані умови завдання

### Основні
- створення файла БД з довільним іменем;
- наповнення БД інформацією з клавіатури;
- вивід записів на екран;
- пошук щонайменше за 3 критеріями: `ID`, `країна`, `дисципліна`;
- доповнення БД новими записами.

### Додаткові
- сортування за очками або результатом;
- зміна запису за `ID`;
- видалення запису за `ID`;
- складний пошук: `дисципліна + макс. час + мін. очки`;
- підрахунок узагальнених характеристик (середній час, середні очки, кількість золотих медалей);
- автозаповнення БД **20 правдоподібними записами**.

## Структура проєкту (модульність через `#include`)
- `main.cpp` — головна програма та меню;
- `database.h` — оголошення функцій модуля БД;
- `database.cpp` — реалізація функцій опрацювання даних;
- `record.h` — структура запису для бінарного файла;
- `Makefile` — збірка проєкту.

## Збірка (Linux/MinGW)

```bash
make
./lab18_variant26
```

## Збірка в Dev-C++ 5.11
1. Створи новий Console Project (C++).
2. Додай файли: `main.cpp`, `database.cpp`, `database.h`, `record.h`.
3. Переконайся, що стандарт компілятора — **GNU++11** або вище.
4. Збери та запусти проєкт.

## Очищення

```bash
make clean
```
